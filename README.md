
combo is a simple program generating all combinations of words you specified.
For example:
```sh
> combo  -n 2 -m 0 password 123 login root
password
123
login
root
123password
loginpassword
rootpassword
password123
login123
root123
passwordlogin
123login
rootlogin
passwordroot
123root
loginroot
```
---
To get help read 'man combo'.
